
Description
-----------
Replaces the Return Scepter with one that only affects the player using it.

Mod options
-----------
Mod options available in config.json:
ReplaceInShop - Whether to replace the regular scepter in the shop or not.
Price - A custom price forÂ  the scepter. -1 will try to copy the original scepter's price or fall back to 2 million.

Install
-------
Install the latest version of SMAPI
Install the latest version of PyTK
Unzip the mod into Stardew Valley/Mods
