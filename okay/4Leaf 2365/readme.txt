
Description
-----------
Gives a player the ability to set their daily luck.

Mod options
-----------
Mod options available in config.json:
Enabled - Whether to override the daily luck or not
Luck - The value for the daily luck. -0.1 is the worst luck, and 0.1 is the best. Values outside this range are clamped.

Install
-------
Install the latest version of SMAPI
Unzip the mod into Stardew Valley/Mods
